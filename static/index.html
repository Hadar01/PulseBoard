Settings & Setup Guide

Connect your tools — step by step

1

Anthropic API Key (Claude LLM)

  1. Go to console.anthropic.com → sign up / log in
  2. Left sidebar → API Keys → Create Key
  3. Copy the key — starts with sk-ant-
  4. Add to .env: ANTHROPIC_API_KEY=sk-ant-xxx
2

Slack Bot Token

  1. Go to api.slack.com/apps → Create New App → "From scratch"
  2. Name it PulseBoard, pick your workspace → Create
  3. Left sidebar → OAuth & Permissions
  4. Under Bot Token Scopes add: channels:history channels:read groups:history users:read
  5. Click Install to Workspace → Authorize
  6. Copy Bot User OAuth Token (starts with xoxb-)
  7. Add to .env: SLACK_BOT_TOKEN=xoxb-xxx and SLACK_CHANNELS=general,engineering
3

GitHub Personal Access Token

  1. Go to github.com/settings/tokens
  2. Click Generate new token (classic)
  3. Select scopes: repo read:org
  4. Copy the token (starts with ghp_)
  5. Add: GITHUB_TOKEN=ghp_xxx and GITHUB_REPOS=YourOrg/YourRepo
4

Notion Integration Token

  1. Go to notion.so/my-integrations → New integration
  2. Name it PulseBoard, enable Read content
  3. Copy the Internal Integration Secret (starts with secret_)
  4. Open your Notion database → ⋯ Add connections → PulseBoard
  5. Copy the database ID from the URL (32-character string after your workspace name)
  6. Add: NOTION_TOKEN=secret_xxx and NOTION_DATABASE_ID=xxx
5

Go Live — turn off Demo Mode

Your .env file should look like this:

DEMO_MODE=false
ANTHROPIC_API_KEY=sk-ant-...
SLACK_BOT_TOKEN=xoxb-...
SLACK_CHANNELS=general,engineering
GITHUB_TOKEN=ghp_...
GITHUB_REPOS=YourOrg/YourRepo
NOTION_TOKEN=secret_...
NOTION_DATABASE_ID=your-32-char-id
DIGEST_INTERVAL_MINUTES=30

Then restart: python main.py
